# alx_html_css

HTML and CSS learning project for ALX.

## Directories

- `html_basic/` - Basic HTML files from previous project
- `css_basic/` - HTML files with CSS styling

## Files

- `html_basic/index.html` - Basic homepage
- `html_basic/tweets.html` - Tweets page
- `css_basic/index.html` - Styled homepage
- `css_basic/tweets.html` - Styled tweets page
- `css_basic/base.css` - Base CSS styling
- `css_basic/styles.css` - Additional styles

## Requirements

- All files end with a new line
- No external libraries (HTML/CSS/JavaScript only)
- W3C compliant code
